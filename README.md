# InterviewAce

InterviewAce is a modern interview preparation platform that helps users practice Frontend, Backend, DSA, and HR interview questions in one place.

The application provides a clean and responsive user experience with authentication, question management, favorites, search, filtering, pagination, dark mode, and more.

---

## Live Demo

Frontend: https://interviewaceprep.netlify.app/

Backend API: https://interview-explorer-backend.onrender.com

---

## Features

### Authentication

- User Registration
- User Login
- User Logout
- Protected Routes

### Question Management

- Add Questions
- Edit Questions
- Delete Questions
- View Detailed Answers

### Favorites System

- Add Questions to Favorites
- Remove Questions from Favorites
- Persistent Storage using Redux Toolkit and Local Storage

### Search and Filtering

- Search Questions
- Filter by Difficulty
- Sort Questions (A-Z / Z-A)

### Pagination

- Browse questions efficiently with pagination support

### User Experience

- Responsive Design
- Dark Mode
- Toast Notifications
- Modern SaaS-Inspired UI

### Categories

- Frontend
- Backend
- DSA
- HR

## Installation

### Clone Repository

```bash
git clone https://github.com/LaxmanRoy14/interview-explorer
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm run dev
```

### Build Frontend

```bash
npm run build
```

---

## Backend Setup

### Install Dependencies

```bash
npm install
```

### Start JSON Server

```bash
npm start
```
